🍫 Chocolate Sales Performance Dashboard

📊 Project Overview

This project is an end-to-end data analysis and dashboard built using Excel.
It analyzes chocolate sales data to generate business insights on revenue, profit, customer behavior, and product performance.

🧹 Data Cleaning

* Removed duplicates and handled missing values (~1% data)
* Replaced null product details with "Unknown"
* Standardized column formats and data types

🔗 Data Integration (Lookups)
  >> Merged multiple tables using Power Query:

    * Products
    * Stores
    * Customers
    * Calendar
 >> Built relationships using product_id, store_id, and customer_id

⚙️ Feature Engineering
  Created calculated columns:

  * Revenue = Quantity × Unit Price
  * Profit = Revenue − Cost
  * Age Group segmentation
  * Month and Day categorization
  * Discount value

📈 Dashboard Features

* KPI Cards:
  * Total Revenue
  * Total Profit
  * Total Orders
  * Average Order Value

* Visualizations:
  * Monthly Revenue Trend
  * Category-wise Performance
  * Revenue by Customer Segment
  * Top 10 Products by Revenue

🎯 Key Insights

* Praline and White chocolate categories generate the highest revenue
* Majority revenue comes from Adult customer segment
* Sales trend shows consistent performance across months
* Top products contribute significantly to total revenue

🛠 Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables
* Data Visualization

📌 Conclusion

This dashboard provides actionable insights into sales performance and helps in data-driven decision making.
